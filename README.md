# Feria Empoderadas y Emprendedoras

Sitio web oficial y dinámico para el evento **Feria Empoderadas y Emprendedoras**, desarrollado sobre WordPress como un tema custom autogestionable y de alta velocidad.

## 🚀 Características Clave

- **Agenda Dinámica e Inteligente:** Alterna automáticamente la pestaña activa (Día 11 o Día 12) según la fecha actual del servidor.
- **Carrusel de Expositoras:** Slider interactivo en el Hero que rota secuencialmente las 15 imágenes de las expositoras reales con controles manuales.
- **Ajustes de Animaciones Premium:** Efectos de hover 3D/paralaje fluidos en las tarjetas de eventos y animaciones *reveal on scroll* optimizadas para las ilustraciones decorativas de pájaros.
- **Estructura del Proyecto:**
  - Carpeta de tema autogestionable: `wp-content/themes/Emprendedora_y_empredendores_brief/`
  - Soporte de configuración de agentes y comandos: `.agents/`, `.claude/`
  - Documentación de control del proyecto: archivos `.md` principales.

## 🛠️ Tecnologías Utilizadas

- **WordPress Core:** CMS base y base de datos relacional para el registro dinámico de eventos.
- **PHP 8.x:** Lógica de plantillas y hooks de inyección automatizada de medios y CPTs en `functions.php`.
- **Vanilla Javascript:** Interactividad libre de jQuery para máxima velocidad de carga.
- **CSS3 Nativo:** Animaciones y transiciones fluidas de alto rendimiento.

## 📁 Archivos Incluidos en este Repositorio

Para mantener el control del tema y evitar almacenar los archivos core de WordPress, este repositorio contiene únicamente:
- La carpeta del tema custom: `/wp-content/themes/Emprendedora_y_empredendores_brief/`
- Los directorios de configuración de desarrollo: `/.agents/`, `/.claude/`
- Los archivos de documentación y registro: `README.md`, `CLAUDE.md`, `AGENTS.md`.
